# Physics Notes

## 何を観察するプロジェクトか

このプロジェクトでは、円柱まわりの流れを 2D で可視化し、Reynolds 数の変化で後流の見え方がどう変わるかを観察することを目的にしています。

主に見たいのは次の 3 点です。

1. 低 Re では後流が比較的落ち着いて見えること
2. Re を上げると、円柱後方で交互渦放出が出やすくなること
3. さらに上げると、実際の流れは 3D 不安定化へ向かうが、このモデルでは 2D の範囲までしか扱わないこと

## Reynolds 数

Reynolds 数は、慣性の影響と粘性の影響の比を表す代表的な無次元数です。このプロジェクトでは、格子上の速度、粘性、円柱径から次式で計算しています。

```text
Re = U D / ν
```

- `U`: 流入速度
- `D`: 円柱径
- `ν`: 動粘性係数に相当する値

ここで扱っている値は格子単位系のパラメータですが、どの値を動かすと Re が上がるかを直感的に確認できるようにしています。

## なぜ 2D なのか

就活用の作品としては、3D CFD を本格的に解くよりも、目的が明確で、UI と計算の関係が説明しやすく、短時間で価値が伝わる構成の方が見せやすいと考えました。

そのため、2D D2Q9 LBM を使って、次のような整理にしています。

- 速度場をリアルタイムに見せる
- 渦度表示で後流の渦構造を見せる
- プローブ信号で周期性を数字として見る
- ただし 3D 乱流そのものは扱わないと明記する

## この作品の見どころ

- 数式と UI をつなぐ題材であること
- 速度表示と渦度表示の両方で観察できること
- プローブ信号を見ながら「見た目」と「時系列」を対応づけられること
- 限界を README / docs で正直に書いていること
