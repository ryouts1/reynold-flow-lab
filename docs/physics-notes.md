# Physics Notes

## 何を見せるための 3D 化か

このプロジェクトの主題は「3D 乱流を厳密に解くこと」ではなく、円柱後流を 3D 格子で扱ったときに、断面の切り方で見える現象がどう変わるかを観察できるようにすることです。

そのため、UI では volume 全体を主役にせず、以下の 2 つを組み合わせています。

- XY / XZ / YZ の断面スライス
- wake の輪郭を点群で見る volume preview

## Reynolds 数

このプロジェクトで表示する Reynolds 数は、以下の簡略式で計算しています。

```text
Re = U * D / ν
```

- `U`: 流入速度
- `D`: 円柱径
- `ν`: 格子上で与える粘性

## 3D ソルバーの扱い

ソルバーは D3Q19 の single-relaxation-time LBM です。

- 流れ方向: `x`
- 断面方向: `y`
- spanwise 方向: `z`

円柱は `z` 方向に貫通した obstacle として置いています。これにより、XY 断面では円柱断面、XZ / YZ 断面では spanwise を含む wake の厚みを観察できます。

## spanwise seed について

低解像度の教育用格子では、完全対称条件のままだと z 方向の差が見えにくくなります。そのため、流入条件に小さな spanwise seed を入れ、XZ / YZ と volume preview で 3D 構造を観察しやすくしています。

この seed は「物理現象を厳密に再現するため」ではなく、「3D の見え方を理解しやすくするため」のパラメータです。
