# Limitations

## 数値モデルの制約

- single-relaxation-time の簡略 LBM を使っています。
- 流入・流出境界は教育用に単純化しています。
- 格子はブラウザで動かす前提のため小さめです。

## 物理的な制約

- 定量的な wake frequency の議論には向きません。
- 高 Reynolds 数の広いレンジを安定に追う用途ではありません。
- spanwise 構造は seed の有無に影響されます。
- 実験や高解像度シミュレーションとの差を詰める実装ではありません。

## 可視化の制約

- volume preview は point cloud ベースの簡易表示です。
- 3D view は見やすさ優先で、ポストプロセスとして点を間引いています。
- slice は瞬時場の観察向けで、定量比較の主役にはしていません。

## 就活ポートフォリオとしての位置づけ

このリポジトリの価値は「3D CFD を完全にやったこと」ではなく、以下をまとめて見せられる点にあります。

- 数値計算を UI に落とし込む力
- worker を使った計算と描画の分離
- 2D ではなく 3D volume を扱う設計判断
- 制約を README と docs で明示する説明力
