# Limitations

## 1. 2D モデルであること

このプロジェクトは 2D D2Q9 LBM を使った教育用の可視化ツールです。円柱後流の 3D 不安定化、mode A / mode B、fully developed turbulence は対象外です。

## 2. 境界条件を簡略化していること

計算を軽く保つために、上端・下端は周期境界、左端は簡易な流入条件、右端は単純な外挿にしています。工学用途の厳密な境界条件設定ではありません。


## 3. 小さな流入擾乱を入れていること

円柱後流が数値的に完全対称な状態へ固定されるのを避けるため、流入境界にはごく小さな時間依存擾乱を加えています。したがって、観察される後流の周期性は純粋な自励振動のみを切り出したものではなく、教育用の見やすさを優先した設定です。

## 4. 定量精度より比較しやすさを優先していること

この作品は「Re を変えると後流がどう見えるか」を直感的に確認することを重視しています。厳密な抗力係数、揚力係数、Strouhal 数の評価には向いていません。

## 5. 高 Re を上げすぎるとモデルの意味が薄れること

スライダー上はある程度高い Re を試せますが、値を上げれば上げるほど、2D 教育用モデルとしての定性的比較という意味合いが強くなります。README や UI でもその前提を崩さないようにしています。

## 6. 可視化作品としての割り切りがあること

この作品は、就活ポートフォリオとして「計算コア」「UI」「説明文書」をひとつにまとめることを重視しています。研究コードや商用解析ソフトの代替を目指していません。
